고용24 회원가입 · 실명인증 · 잡케어 4 STEP

↓

고용24는 모든 온라인 고용서비스를 한 곳에서 신청하고 결과를 확인할 수 있는 통합 포털입니다. 빌드업 · 점프업 참여 전에 미리 가입해 두면 상담 시간이 훨씬 줄어듭니다.

1

고용24 회원가입
고용24 사이트에서 상단 ‘회원가입’ 클릭 → ‘개인회원가입’ 선택 → ‘14세 이상 개인회원’ 선택 후
본인확인 방법(카카오, 네이버, 휴대폰 등)을 선택해 인증하고, 기본 정보 입력 후 회원가입을 완료합니다.

2

첫 로그인 후 실명인증(최초 1회)
상단 ‘로그인’ 클릭 → 방금 가입한 계정으로 로그인 또는 간편인증 로그인 →
화면 상단 메뉴에서 ‘실명인증’ 버튼 클릭 → 주민등록번호와 성명 입력 후 ‘실명확인’을 눌러 인증합니다.
※ 실명인증을 한 뒤에는 한 번 로그아웃될 수 있으므로, 다시 로그인해야 모든 서비스 이용이 가능합니다.

3

(선택) 공동인증서 등록
공동인증서로 로그인하고 싶다면, 로그인 후 상단 메뉴에서 ‘공동인증서 등록’을 클릭합니다.
아이디와 비밀번호를 입력하고, 사용 중인 인증서를 선택해 공동인증서를 등록합니다.
※ 기존 워크넷·직업훈련 등에서 쓰던 인증서는 사용되지 않으며, 고용24 홈페이지에서 새로 등록해야 합니다.

4

잡케어 시작하기
로그인 후 메인 화면에서 [취업준비] 아이콘을 누르고, 화면에 보이는 ‘잡케어 시작하기’ 버튼을 선택합니다.
전공, 관심 직무, 배우고 싶은 기술 등을 키워드로 입력하고 직업 추천 받기를 누른 뒤,
마음에 드는 직업을 선택해 ‘직업 설계서 만들기’ 버튼으로 저장하면 완료입니다.

※ 본인확인은 주민등록번호를 직접 입력하지 않고 본인임을 확인하는 절차이고, 실명인증은 온라인 민원·신청 등에서 주민등록번호를 대신해 사용하는 전자서명 성격의 인증입니다. 실명인증은 최초 1회만 하면 됩니다.

    나를 이해하기 최근 재미있었던 활동, 강점, 성격 등을 정리하며 진짜 ‘나’를 돌아봅니다.
